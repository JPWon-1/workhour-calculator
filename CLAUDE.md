# 출퇴근 / 근무시간 계산 룰

사용자 회사의 근무시간 계산 규칙. 출퇴근정보 xlsx 분석 시 항상 이 규칙을 적용한다.

## 출근 라운딩 (올림, 30분 단위)
- 어떤 시각에 찍든 무조건 **다음 30분 시작**으로 처리됨
  - 08:29 → 08:30 시작
  - 08:31 → 09:00 시작
- 정시 출근 시 항상 **08:30 시작 고정**

## 퇴근 캡 (상한 18:00)
- 18:00 이후 찍어도 인정되지 않음
  - 18:02 → 18:00 처리
- 시간외근무 별도 신청 없으면 일일 최대 정산 종료시각 = **18:00**

## 정산시간 계산
- 정산시간 = (라운딩된 퇴근시각) − (라운딩된 출근시각) − 점심 1시간
- **일일 최대 정산시간 = 8.5시간** (08:30 ~ 18:00, 점심 1h 차감)

## 연차 계산 (시간 단위 절삭)
- 연차/시간 인정이 **시간 단위로 내림** 처리됨
- 1분이라도 모자라면 1시간을 통째로 못 받음 (예: 평균 7.9995h → 7시간만 인정)
- 따라서 평균이 정수 시간에서 살짝 미달하는 게 확정이면, 그 시간은 어차피 손해 → 차라리 일찍 퇴근하는 편이 합리적

## 근무 형태
- 표준 근무시간: **1일 8시간** (`선택근로8`)
- 점심: 1시간 차감
- 외근(종일): 8시간 정산
- 외근(오후): 4시간 + 오전 출근분

## 사용자 정보
- 소속: 그물코사업처 / 그린사업국 / 그린마케팅팀
- 직급: 단기활동가 (월급제)

## 분석 도구
- 스크립트: `workhour.py`
- 사용법: `python3 workhour.py "<xlsx 경로>"`
